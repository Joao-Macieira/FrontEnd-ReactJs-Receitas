## Instalação
Você pode clonar este repositório OU baixar o .zip


## Configuração
Todos os arquivos de **configuração** e aplicação estão dentro de *src/index.js*.
Todas as **rotas** da aplicação estão na pasta *src/routes/index.js*.
As configurações de **autenticação** estão na pasta *src/context*.

## Uso
Basta **iniciar o servidor** com o comando *npm start* ou *yarn start*
